| [Анализ пользовательского поведения в мобильном приложении](09_analysis_of_user_behavior) | Анализ воронки продаж и результатов A/A/B-тестирования.| *pandas*, *matplotlib*, *numpy*, *plotly*, *seaborn*, *scipy* |
